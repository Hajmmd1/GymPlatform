# GymPlatform

A modern, cloud-native gym management SaaS platform.

## Overview

GymPlatform is an enterprise-grade solution for gym management, featuring:
- Member management and billing
- Class scheduling and booking
- Trainer portals and payments
- Multi-location support

## Technology Stack

### Backend
- .NET 8 (ASP.NET Core)
- PostgreSQL
- Redis
- RabbitMQ

### Frontend
- Next.js 15
- React 19
- TypeScript
- Tailwind CSS

### Mobile
- React Native
- Expo SDK
